Buscar conteudo

UMLUseCase
Projeto 2 :: Model :: Buscar conteudo

Description

Fluxo principal:

  1. Usuário acessa o sistema e solicita um conteúdo (F001)

  2. Usuário recebe resposta da solicitação (F002)

(F001) Fluxo alternativo - buscar filtros:

  1. Usuário fornece opções de filtro

  2. Retorna ao passo 2 do fluxo principal

(F002) Fluxo alternativo - sem resposta:

  1. Usuário recebe resposta de que o conteúdo buscado não foi encontrado
