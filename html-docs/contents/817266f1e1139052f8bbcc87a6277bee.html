Solicitar conteúdo

UMLUseCase
Projeto 2 :: Model :: Solicitar conteúdo

Description

Fluxo principal:

  1. Recebe requisição de conteúdo

  2. Converte a URL para um IP atrvés de um servidor DNS

  3. Busca o conteúdo solicitado (F001, F002, F003)

  4. Retorna o conteúdo solicitado como resposta (F004)

(F001) Fluxo alternativo - busca HTTP:

  1. Arquivos HTTP são carregados como respota

  2. Retorna ao passo 3 do fluxo principal

(F002) Fluxo alternativo - busca FTP:

  1. Documentos são carregados como respota

  2. Retorna ao passo 3 do fluxo principal

(F003) Fluxo alternativo - busca SMTP:

  1. Conteúdo dos emails são carregados como respota

  2. Retorna ao passo 3 do fluxo principal

(F004) Fluxo alternativo - conteúdo não encontrado

  1. Conteúdo não é encontrado

  2. Um erro é retornado como resposta
